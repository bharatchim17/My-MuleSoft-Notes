 //Auto discovery:-

 We use API Autodiscovery to pair an API in API Manager to its deployed Mule application.
 Is the process of allowing API manager application to discover the application is running in
 the runtime using instance , ID is known as auto discovery..
 To enable auto discovery follow below steps:
 1. Create api autodiscovery global element using API instance id & select main flow of API kit router..
 2. provide below runtime properties:
  - Organization buisness group client_id
  - Organization buisness group client_secret
 
global element----> API Autodiscovery---> api id: provide instance here | You can also
 choose to use the ${apiId} value and reference it from a config.yaml file.
   
flow Name: API kit router flow i.e. Main flow not xml file
